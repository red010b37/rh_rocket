Sass

Map

RGB color produced from separated SASS Maps: 

$utilities-bg-colors: map-loop($utilities-bg, rgba-css-var, "$key", "bg");

Utilities API

Utilities Background created from scss/components/bootstrap/_utilities.scss

"background-color": (
    property: background-color,
    class: bg,
    local-vars: (
        "bg-opacity": 1
    ),
    values: map-merge(
        $utilities-bg-colors,
        (
        "transparent": transparent
        )
    )
),
