Sass

Map

RGB color produced from separated SASS Map: 

$utilities-text-colors: map-loop($utilities-text, rgba-css-var, "$key", "text");

Utilities API

Utilities Background can be found onscss/components/bootstrap/_utilities.scss

"color": (
    property: color,
    class: text,
    local-vars: (
        "text-opacity": 1
    ),
    values: map-merge(
        $utilities-text-colors,
        (
        "reset": inherit,
        )
    )
),
"text-opacity": (
    css-var: true,
    class: text-opacity,
    values: (
        17: .17,
        25: .25,
        33: .33,
        50: .5,
        75: .75,
        100: 1
    )
),
